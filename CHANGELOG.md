### 1.0.0 - Initial Release
FreeBooks module for reading EPUB format books in the Open Neom ecosystem.

Key Features:
- EPUB book reader with `epub_view` integration
- XML to JSON parsing for book data
- Local book storage with ObjectDB
- Spinner loading states with FlutterSpinKit
- Toast notifications with Fluttertoast
- Network requests with Dio

Leverages Core Modules:
- Built upon `neom_core` for foundational services
- Uses `neom_commons` for shared UI components
