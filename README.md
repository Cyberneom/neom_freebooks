# neom_freebooks

Free ebook browser and reader for the Open Neom ecosystem.

Provides EPUB reading with downloading, bookmarking, genre browsing, favorites, and library exploration.

## Features
- EPUB format reading
- Book downloading and offline access
- Bookmarking
- Genre and category browsing
- Favorites library
- Library exploration

## Dependencies
- `neom_core` — Core services
- `neom_commons` — Shared UI components
